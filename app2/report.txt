
App 2 was implemented using an array (size 3000 corresponding to the maximum ID# value), where each index of the array corresponds to the a student ID#, and the index stores the respective student's age. This allows for O(1) lookup, insertion, deletion (via direct indextong while also keeping code extremely simple and readable (as opposed to a hash map implementation).

Time Complexity of Functions

createDataSet: O(n) 

destroyDataSet: O(1)

searchID: O(1)

insertion: O(1)

deletion: O(1)
